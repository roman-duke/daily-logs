I like the teaching style of the Grokking Relational Database Design. It is a nice blend of using practical applications to quickly explain the database design concepts and using fun analogies to strengthen concepts; it's quite pedagogical. I chose this book because I was pressed for time and still wanted a medium to grasp the basic concepts of relational database design. The author of the book also states how in this modern age (the age of generative AI), learning has never been faster. chatGPT is a relatively reliable way of getting quick, well tailored answers that would've otherwise been gotten via a more involving iterative process of searching the web. But it is to be used with caution, because AI can be quite confident even when wrong.
